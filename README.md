# Hackerank/LC/Neetcode
This repository is a collection of solutions and practice implementations for various Data Structures and Algorithms (DSA) problems, primarily sourced from platforms like HackerRank and LeetCode.

The goal of this repository is to track progress, document optimized approaches, and provide a quick reference for common algorithmic patterns.

📂 Repository Structure
The solutions are implemented in Python and cover a wide range of topics:

🧩 Problem Categories
Arrays & Strings: Includes operations like array reversal, left rotation, and handling sub-arrays.

Data Structures: Custom implementations and usage of structures like Singly Linked Lists.

Dynamic Programming: Solutions for problems like Longest Increasing Subsequence (LIS).

Intervals: Merging, overlapping, and manipulation of time/data intervals.

Two Pointers & Sliding Window: Optimized techniques for searching and processing data efficiently.

🛠 Notable Files2_pointers.py: Implementations using the two-pointer technique.sliding_window.py: Solutions utilizing the sliding window pattern for optimal time complexity.Leetcode_300_nlogn.py: An optimized $O(n \log n)$ approach for the Longest Increasing Subsequence problem.merge_overlap_intervals.py: Algorithm to efficiently merge overlapping intervals.SinglyLinkedListNode.py: Fundamental structure for linked list operations.
